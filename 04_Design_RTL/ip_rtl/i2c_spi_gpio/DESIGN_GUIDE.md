# 低速外设设计指南

## 1. 概述

低速外设包括GPIO、I2C、SPI、UART等基本外设接口。

## 2. 外设清单

| 外设 | 数量 | 描述 |
|------|------|------|
| GPIO | 64-128 | 通用输入输出 |
| I2C | 8 | 双线串行接口 |
| SPI | 8 | 同步串行接口 |
| UART | 8 | 异步收发器 |
| Timer | 16 | 可编程定时器 |
| Watchdog | 2 | 系统监控 |

## 3. 开发任务

1. APB桥接器（2周）
2. GPIO控制器（2周）
3. I2C控制器（3周）
4. SPI控制器（3周）
5. UART控制器（2周）
6. Timer/Watchdog（3周）
