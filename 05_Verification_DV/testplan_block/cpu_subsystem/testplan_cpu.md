## 1. 模块概述

### 1.1 功能描述
CPU 子系统包含 Cortex-A78AE 应用处理器和 GIC-600 中断控制器。

### 1.2 验证范围
- Cortex-A78AE Core
- L1/L2 Cache
- GIC-600 中断控制器
- AMBA 总线接口
