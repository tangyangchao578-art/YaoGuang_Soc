## 1. 性能指标

### 1.1 核心规格

| 指标 | 目标值 | 备注 |
|------|--------|------|
| 工艺节点 | TSMC 7nm FinFET | 成熟工艺 |
| 核心电压 | 0.75V - 1.0V | DVFS 支持 |
| NPU 算力 | ≥ 300 TOPS (INT8) | 4x Clusters |
| CPU 性能 | ≥ 100K DMIPS | 8x ARM Cortex-A78AE |
| GPU 性能 | ≥ 2 TFLOPS (FP32) | Mali-G78 或自研 |
| 目标频率 | 1.5 - 2.0 GHz | CPU/GPU/NPU |
| 动态功耗 | ≤ 55W | 典型工作负载 |
| 静态功耗 | ≤ 5W | 待机状态 |
| 峰值功耗 | ≤ 60W | 最大工作负载 |
| 面积预算 | ≤ 120 mm² | 包括所有模块 |
| 单片成本目标 | ≤ $333 | 量产 100K 片/年 |

### 1.2 功能要求

#### 核心功能
- [x] NPU AI 加速（300+ TOPS INT8）
- [x] 多核 CPU 应用处理（8x Cortex-A78AE）
- [x] GPU 通用计算加速（2x Cores）
- [x] ISP 图像处理（16+ 路摄像头）
- [x] LPDDR5 内存控制器（32-64 GB）
- [x] NoC 片上互联网络
- [x] Safety Island 安全机制（ASIL-D）
- [ ] 外设接口（MIPI, PCIe, Ethernet, USB, CAN FD）
- [ ] 电源管理（DVFS, 多电源域）

#### 性能要求
- [ ] NPU 延迟 ≤ 10ms (单次推理)
- [ ] 内存带宽 ≥ 200 GB/s
- [ ] NoC 峰值带宽 ≥ 500 GB/s
- [ ] 系统频率 ≥ 1.5 GHz
- [ ] 能效比 ≥ 5 TOPS/W
